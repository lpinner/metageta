 22  ''' 
 23  Image Metadata drivers 
 24  ====================== 
 25  Generic and custom image format drivers 
 26   
 27  B{Example}: 
 28      >>> import metageta.formats 
 29      >>> dataset = metageta.formats.open(somefile) 
 30      >>> print dataset.extent 
 31      [[139.214834,-20.155611],[139.28967,-20.170712],[139.403241,-19.311661],[139.47766,-19.326724]] 
 32      >>> print dataset.metadata['filetype'] 
 33      CEOS/Landsat CCRS Format 
 34      >>> print dataset.metadata['datatype'] 
 35      Int16 
 36   
 37   
 38  B{Adding support for another format}: 
 39      - Create a new .py file. Name the .py file whatever you want as long as the filename doesn't begin with an underscore "_" 
 40      - Define one or more "regular expression" strings to match your formats filename 
 41      - Create a "Dataset" class that inherits either from the base __dataset__.Dataset class or from the __default__.Dataset class (See class hierarchy below) 
 42      - The default Dataset class is useful if GDAL can read your format and you just need to populate some extra fields. 
 43      - Populate appropriate filelist & fileinfo in the __init__ method of your dataset class (if required), do not populate the metadata dict! 
 44      - Populate appropriate metadata and extent variables in the __getmetadata__ method of your dataset class 
 45      - Your format will be automatically loaded when the formats module is initialised. 
 46      - Errors should be propagated back up the chain. If you can't handle a certain file and for some reason you don't want an error to get raised (eg. the ENVI driver (*.hdr) doesn't handle ESRI bil/flt headers (*.hdr)) then raise NotImplementedError which will be ignored in lib.formats.Open() 
 47      - If you want some info to get logged by the application and then continue processing (e.g the image doesn't have a projection defined, etc...) then use the warnings.warn("Some message") method - don't forget to import the warnings module! 
 48      - Date/Time formats must follow follow AS ISO 8601-2007 (see: U{http://www.anzlic.org.au/metadata/guidelines/Index.html?date_and_datetime.htm}) 
 49
