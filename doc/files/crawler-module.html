Iterator for metadata crawling.

Example:

>>> import crawler
>>> Crawler=crawler.Crawler('Some directory to crawl')
>>> #Loop thru dataset objects returned by Crawler
>>> for dataset in Crawler:
>>>     metadata=dataset.metadata
