Read Metadata for an ACRES Landsat FastL7A format image as GDAL doesn't get it all. Format description: http://www.ga.gov.au/image_cache/GA10348.pdf

Note: hrf = ~30m VNIR/SWIR (bands 1-5 & 7) htm = ~60m thermal (band 6) hpn = ~15m pan (band 8)
